# Real-Time IC Piston Defect Detection using YOLOv11

## 📌 Overview
This project presents an AI-powered computer vision system designed to detect defects in internal combustion (IC) engine pistons in real time. The system uses the YOLOv8 deep learning object detection model to automatically identify manufacturing defects such as cracks, scratches, dents, and porosity on piston surfaces.

The goal of this project is to improve industrial quality control by replacing manual inspection with an automated, fast, and accurate defect detection system.

---

## 🚀 Features
- Real-time defect detection using YOLOv11
- Detection of multiple piston defects
- High-speed image processing
- Automated quality inspection
- Scalable for industrial manufacturing lines

---

## 🔍 Defects Detected
The model is trained to detect the following piston defects:

- Crack
- Scratch
- Dent
- Surface Porosity
- Ring Groove Damage

These defects are critical because they can affect engine performance, efficiency, and durability.
